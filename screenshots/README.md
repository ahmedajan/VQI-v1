# Screenshots

Add application screenshots here before publishing:

1. `main_window.png` -- Full application window with score gauges displayed
2. `feedback_summary.png` -- Summary feedback tab showing plain-language recommendations
3. `feedback_expert.png` -- Expert details tab with feature-level diagnostics
4. `waveform_view.png` -- Waveform, spectrogram, and mel spectrogram visualization

To capture screenshots:
1. Run VQI.exe (or `python -m vqi.gui.app` from the development repo)
2. Score an audio file
3. Use Windows Snipping Tool (Win+Shift+S) to capture each view
4. Save as PNG files in this directory
